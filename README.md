# Chat Assistant — ProxyAPI

Консольный ассистент для обмена сообщениями через ProxyAPI с поддержкой двух режимов работы и сохранением истории диалога.

## Возможности

- **Два режима работы:**
  - 🧠 **Думающая модель** (Claude Sonnet 4.5) — с отображением цепочки рассуждений (reasoning) и точных метрик токенов: входные, выходные, всего
  - 💬 **Обычная модель** (GPT-5.4) — быстрый ответ через OpenAI-совместимый Chat Completions API
- **Сохранение и загрузка истории** — диалог сохраняется в `history.json` и подгружается при следующем запуске
- **Обработка ошибок и таймаутов** — при сбое запроса выводится понятное сообщение, сообщение пользователя откатывается из истории
- **Понятные логи запуска** — при старте показывает, загружена история или новый диалог, текущий режим, количество сообщений при выходе
- **Команды:**
  - `exit` / `quit` — выход с сохранением истории
  - `clear` — очистка истории

## Установка

1. Клонируйте репозиторий и перейдите в директорию:
```bash
cd AI_Assistant
```

2. Создайте и активируйте виртуальное окружение:
```bash
python -m venv .venv
source .venv/bin/activate   # macOS/Linux
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Создайте файл `.env` на основе `.env.example` и укажите API-ключи:
```env
ANTHROPIC_API_KEY=ваш_ключ
OPENAI_API_KEY=ваш_ключ
```

## Запуск

```bash
python chat.py
```

## Конфигурация

Все настройки хранятся в файле `.env`:

| Переменная | Описание |
|---|---|
| `ANTHROPIC_API_KEY` | API-ключ для Anthropic (ProxyAPI) |
| `OPENAI_API_KEY` | API-ключ для OpenAI (ProxyAPI) |

## Зависимости

| Пакет | Назначение |
|---|---|
| `anthropic` | Клиент для Claude (думающая модель) |
| `openai` | Клиент для OpenAI Chat Completions API |
| `python-dotenv` | Загрузка переменных из `.env` |
